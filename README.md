# JARVIS — Embedded AI Voice Assistant

Embedded AI voice assistant built on Raspberry Pi Zero 2W — wake-word detection, LLM inference, STT/TTS, I2S audio, and real-time sensor integration.

## Tech Stack
Raspberry Pi Zero 2W · Python · Linux · Vosk · Whisper API · I2S Audio · GPIO · SPI TFT Display · HC-SR04
